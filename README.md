# s7-proyect-conectaTel
📱 ConnectaTel - Análisis de Comportamiento de Usuarios
🎯 Descripción del Proyecto
Análisis completo del comportamiento de usuarios de telecomunicaciones para ConnectaTel, enfocado en entender patrones de consumo según edad, volumen de llamadas/mensajes y nivel de uso. Este proyecto proporciona insights accionables para optimizar ofertas comerciales y estrategias de segmentación.

📊 Datasets Analizados
users_latam: Información demográfica de usuarios (edad, ubicación)
usage: Datos de uso real (llamadas, mensajes, minutos)
plans: Información de planes de servicio
🔧 Tecnologías Utilizadas
- Python 3.x
- Pandas & NumPy (manipulación de datos)
- Matplotlib & Seaborn (visualización)
- Jupyter Notebook (desarrollo)
📈 Metodología
Exploración y Carga de Datos
Identificación de Problemas de Calidad
Limpieza y Tratamiento de Datos
Análisis Estadístico Descriptivo
Visualización y Detección de Outliers
Segmentación de Usuarios
Insights y Recomendaciones Comerciales
🎯 Resultados Clave
Identificación de patrones de uso por segmento demográfico
Tratamiento de outliers mediante winsorización
Segmentación accionable para estrategias comerciales
Recomendaciones para optimización de planes
